  • <frame> tag

    This tag was used inside a <frameset> to define a single frame window. Deprecated in HTML5.

  • <frameset> tag

    Replaced by CSS and <iframe>. It was used to divide browser window into multiple frames.

  • <nobr> tag This text will not break into a new line using <nobr>. We should not use derecated tags inside our project as it is not a good practice although browser may support them. We should use alternative ways recomended by senior developers and official documentation of that language.

  • <plaintext> tag

    This tag displayed all following content as plain text, including tags. Now replaced by <pre> or escaping HTML.

  • <rb> tag

    Used in Ruby annotations (East Asian typography), but not widely supported and now deprecated.

  • <rtc> tag

    Part of Ruby annotation structure; provided extra semantic information. Deprecated due to low support.

  • <strike> tag This text is struck through using <strike> tag.
  • <tt> tag This text appears in teletype (monospace) font.
  • <xmp> tag

    The <xmp> tag in HTML was used a long time ago to display code or text exactly as it is, including HTML tags without the browser interpreting them. It just tells the browser: "Just show everything inside me as plain text, not HTML."
